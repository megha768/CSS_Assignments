What is CSS?

  1. CSS stands for Cascading Style Sheet.
  2. CSS describes how HTML elements are to be displayed on the screen.
  3. CSS is not necessary CSS saves a lot of work.It can control the layout of multiple webpages all at once.
  4. External StyleSheet are stored in CSS files.

CSS can be added to HTML documents in 3 ways :

  • Inline - by using the style attribute inside HTML elements.
  • Internal - by using a style element in the head section.
  • External - by using a link element to link to an external CSS file.
  • The most common way to add CSS is to keep the styles in the external CSS files.However, in this tutorial we will use inline and internal styles because this is easier to demonstrate and easier for you to try it yourself.

When comparing CSS class vs Id, the difference is that CSS class applies a style to multiple elements, Id on the other hand applies a style to one unique element. Id is also special in that you can use a special URL to link directly to an element and it's used by JavaScript.

In CSS, selectors are used to target a specific element or range of elements on a webpage. Once element has been targeted, a style or set of styles can be applied to the element.

There is a wide range of selectors available. Two of the most commonly used are class and Id. Both are used to target the elements to which a style should be applied.
